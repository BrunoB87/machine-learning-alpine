# Alpine Container for Machine Learning

## What is include
- Python 3.5
- NumPy
- SciPy
- Scikit-learn
- Pandas
- Matplotlib
- Seaborn
- XGBoost
- Jupyter Notebook


All in only ~200 MB :)

## Running the container
Run `docker-composer up` and open your browser in `http:localhost:8888`.

See the `Welcome.ipynb` to all versions.


Enjoy :)
